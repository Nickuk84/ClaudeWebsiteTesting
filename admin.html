Welcome to the CMS

Manage your website content and publish changes directly to your live site — no coding required.

📝 Edit Content

Use the News, Team, and Careers tabs to add, edit, or remove content. Changes save automatically to your browser.

👀 Preview Changes

Click the Preview tab to see how your site looks right now. Changes to config are reflected when you refresh the preview.

🚀 Publish to Live Site

  1. Make your edits in the content tabs
  2. Go to Publish tab
  3. Connect your GitHub token (one time)
  4. Click Publish Now
  5. Live site updates in ~60 seconds!

💾 Backup & Restore

Use Settings → Export JSON to back up all your content. Import it later to restore or transfer to another browser.
